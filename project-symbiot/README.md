# Project Symbiot — Unified Omnichannel Protocol

This repo contains:
- **backend/**: FastAPI modular kernel (NeuroSync, QuantumLink, Fee Sweeper)
- **frontend/**: static dashboard (index.html) calling backend APIs

## Quickstart
```bash
bash scripts/dev.sh
```

Backend: http://127.0.0.1:8000
Docs: http://127.0.0.1:8000/docs
Frontend: open frontend/index.html in your browser

Environment
	•	Python 3.10+ recommended
	•	Optional: uv for faster dependency install
